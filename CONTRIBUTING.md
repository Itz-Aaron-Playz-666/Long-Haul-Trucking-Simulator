# Contributing to Truck Simulator

Thank you for your interest in contributing! 🚚

## How to Contribute
1. **Fork the repository** and create your branch from `main`.
2. **Make your changes** (fix bugs, add features, improve docs).
3. **Test your changes** to ensure they work across platforms.
4. **Submit a Pull Request** with a clear description of your changes.

## Coding Guidelines
- Use **C# .NET 8** for all code.
- Follow consistent naming conventions (`PascalCase` for classes, `camelCase` for variables).
- Keep commits small and descriptive.

## Reporting Issues
- Use the **Issues tab** to report bugs or suggest features.
- Include steps to reproduce problems when possible.

## Community
- Be respectful and constructive.
- Help beginners — this project is kid‑friendly and learning‑focused.
